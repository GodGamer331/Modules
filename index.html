Roblox -> Discord Webhook API

This module aims to make making webhooks to discord easy and customizable.
Download the module here.
Module Version: v1.0.0-a


Requiring the module and sending webhook:

                
local WebhookAPI = require(script.Parent) --Your directory here.
local NewWebhook = WebhookAPI.new()
-- ...
NewWebhook:Send(WebhookURL)

Base functions



.new()

This is to create a new Webhook object.
Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()



25 LIMIT
:AddEmbed()

This function is used to create a embed within the webhook.
Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new()
NewWebhook:AddEmbed()



:RemoveEmbed(EmbedNumber: number)

This function removes a embed from the embed list if it exists.
Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new()
NewWebhook:AddEmbed()
NewWebhook:AddEmbed()

NewWebhook:RemoveEmbed(1)   --> the 2nd embed will be put into the 1st position after.



25 LIMIT
:AddField(EmbedNumber: number, name: string, value: string, inline: string)

This function allows you to add up to NUMBER fields to your embed.
Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new()

local FieldName = "Field Name!"
local FieldValue = "Value!"
local inline = false    -- Optional, can be nil.
local EmbedNumber = nil -- Can be nil, or any number.

NewWebhook:AddEmbed()
NewWebhook:AddField(EmbedNumber, FieldName, FieldValue, inline)
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end

Events



.Changed

This event fires any time a change is made to the webhook.

:Connect(function(...))


:Wait() YIELDS


:Once(function(...))


Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()

NewWebhook.Changed:Connect(function(...)
    local T = {...} -- This is preffered way of using .Changed event as the amount of values vary.
    print(#T)        --> X values
end)

--. . .

NewWebhook:AddEmbed()
--. . .


--. . .
NewWebhook.Changed:Wait()
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end

.Sent

This event is fired when you send a webhook.

:Connect(function(WebhookData: {}))


:Wait() YIELDS


:Once(function(WebhookData: {}))


Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()

NewWebhook.Sent:Connect(function(Data)
    print(Data)     --> {["embeds"] = {. . .}, . . .}
end)

--. . .

NewWebhook:AddEmbed()
--. . .


--. . .
NewWebhook.Changed:Wait()
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end

Simple Utils

This module aims to house a lot of helpful functions.
You can download the module here.
Module Version: v1.2.0

Requiring the module


local Utils = require(YourPathHere) -- You can require the entire module
local Invert = require(Utils.Invert) -- or you can require the functions alone.

Base Functions



.ConvertTime(Seconds: number)

This function allows you to put seconds into minutes:seconds (00:00 format). 

local Utils = require(YourPathHere)
local Seconds = 85

print(Utils.ConvertTime(Seconds))   --> "01:25"


Gates

This function includes several logic gates you may use as well as :GetTruthTable();

AND | OR | XOR | NAND | NOR | XNOR


local Utils = require(YourPathHere)

local TruthTable = Util.Gates:GetTruthTable()
print(TruthTable)       --> All inputs, and outputs for the gates.

print(Utils.Gates.OR(true, false))   --> true


.GetDescendantsOfClass(Ancestor: Instance, Class: ClassName?)

This functions gets all descendants of Ancestor which have the ClassName Class. 

local Utils = require(YourPathHere)

local Descendants = Utils.GetDescendantsOfClass(workspace, "Sound")
print(Descendants)      --> {Instance}


.GetPath(Base: Instance)

This function returns the entire path leading up to that instance as a string. 

local Utils = require(YourPathHere)
local Part = game.Workspace.Part

print(Utils.GetPath(Part))      --> "game.Workspace.Part"


.Invert(Num: number)

This function inverts any number, 0 will always return 0. 

local Utils = require(YourPathHere)

print(Utils.Invert(123), Utils.Invert(-456), Utils.Invert(0), Utils.Invert("789"), Utils.Invert("A"))   --> -123, 456, 0, -789, nil


.Reverse(Thing: number | string)

This function will reverse any number or string. 

local Utils = require(YourPathHere)

print(Utils.Reverse(123), Utils.Reverse(-987), Utils.Reverse("Hello"))   --> 321, -789, "olleH"


.Round(Num: number, Decimal: number)

This function will round the number down to the nearest digits. 

local Utils = require(YourPathHere)

print(Utils.Round(23.587, 2), Utils.Round(2587.159753, 4), Utils.Round("Abc 12.157", 2))   --> 23.58, 2587.1597, nil


.Shuffle(T: {number | string | {. . .}})

This function will shuffle the table randomly. 

local Utils = require(YourPathHere)
local Table = {1, 5, 7, 8, 11, 15}

print(Utils.Shuffle(Table))   --> {7, 5, 15, 1, 11, 8}


.Sort(ToSort: {number}, SortBy: string)

This table will sort numbers to be in order. 

local Utils = require(YourPathHere)

print(Utils.Sort({1, 88, 5, 51, 28, 15}, "htl"), Utils.Sort({1, 88, 5, 51, 28, 15}, "lth"))   --> {88, 51, 28, 15, 5, 1}, {1, 5, 15, 28, 51, 88}


.Transform(ToTransform: number | string)

This function will return a number in more readable way. 

local Utils = require(YourPathHere)

print(Utils.Transform(123456789), Utils.Transform("54678"))   --> "123,456,789", "54,678"


.Signal. ...

This set of functions allows you to create a custom RBXScriptSignal object.
Its best used within other modules for events.
The module has 2 main sub functions, and all the RBXScriptSignal functions (:Connect, :Once, :Wait, . . .).
.new() -> Creates a new RBXScriptSignal.
:fire(Anything) -> Fires all the functions connected to the event. 

local Utils = require(YourPathHere)

local CustomSignal = Utils.Signal.new()

CustomSignal:Connect(function(...)
    local Received = {...}
    print(Received)     --> {...any}
end)

CustomSignal:fire("Any amount of arguments here.", true, game.Workspace.Part)
