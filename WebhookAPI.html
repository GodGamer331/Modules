Roblox -> Discord Webhook API

This module aims to make making webhooks to discord easy and customizable.
Download the module here.
Module Version: Version not loaded?


Requiring the module and sending webhook:

                
local WebhookAPI = require(script.Parent) --Your directory here.
local NewWebhook = WebhookAPI.new()
-- ...
NewWebhook:Send(WebhookURL)

Base functions



.new(url)

This is to create a new Webhook object.
Key Type Required Default
url string true nil

Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new("https://discord.com/api/webhooks/xxxxxxxxxxxxxxxxxxx/fYtwNxxyzfsXmaxxR0R5nEvxxx623OR8xxJNy8I0omvKAyezIxxo2hV04bXDkmtTgQxx")



:SetMessage(Message)

This function is used to create a embed within the webhook.
Key Type Required Default
Message string true nil

Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new(webhook_url)
NewWebhook:SetMessage("This is a test message!")



:SetProfile(username, avatar_url)

This function removes a embed from the embed list if it exists.
Key Type Required Default
username string false nil
avatar_url string false nil

Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new(webhook_url)
NewWebhook:SetAvatar("A test name", "Random google image...")



:AddAttachments(Data)

This function allows you to add attachments to the embed/message.
Key Type Required Default
Data
Key Type Required Default
number
Key Type Required Default
description string false nil
filename string true nil
filecontent string true nil
file_extension string true nil
true nil
true nil

Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new(webhook_url)

local FileName = "File Name!"
local Content = "This is a .txt file!"

NewWebhook:AddAttachment({[1] = {description = nil, filename = FileName, filecontent = Content, file_extension = "txt"}})
local success,err = NewWebhook:Send()

if not success then
    warn(err)
end

:SetAuthor(Name, Avatar_url, click_url)

This function sets the author, which is at the very top of the embed.
Key Type Required Default
Name string true nil
Avatar_url string false nil
click_url string false nil

Example 

local WebhookAPI = require(script.Parent)
    
local NewWebhook = WebhookAPI.new()

--. . .

local avatar_url = "https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRd-1PwCVQptnQcktkXKCE5_ZtA0e1qmb5ZDPRC8NLy4t7BuDjYGzeQYPzNAXQVn1p8cVQ&usqp=CAU"

local emb = API.NewEmbed()
--. . .
emb:SetAuthor("I'm a Author!", avatar_url, "https://google.com")

NewWebhook:AddEmbed(emb)

local success,err = NewWebhook:Send()

if not success then
    warn(err)
end

:Send(webhook_url, data)

Sends the webhook to discord.
Key Type Required Default
webhook_url string false nil
data table false nil

Example 

local WebhookAPI = require(script.Parent)

local success,err = WebhookAPI:Send(webhook_url, {"WEBHOOK DATA HERE"}) -- You do NOT need to put webhook url, OR webhook data if your used the .new constructor.

if not success then
    warn(err)
end


Events



.Changed

This event fires any time a change is made to the webhook.

:Connect(function(...))


:Wait() YIELDS


:Once(function(...))


Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()

NewWebhook.Changed:Connect(function(...)
    local T = {...} -- This is preffered way of using .Changed event as the amount of values vary.
    print(#T)        --> X values
end)

--. . .

NewWebhook:AddEmbed()
--. . .


--. . .
NewWebhook.Changed:Wait()
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end

.Sent

This event is fired when you send a webhook.

:Connect(function(WebhookData: {}))


:Wait() YIELDS


:Once(function(WebhookData: {}))


Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()

NewWebhook.Sent:Connect(function(Data)
    print(Data)     --> {["embeds"] = {. . .}, . . .}
end)

--. . .

NewWebhook:AddEmbed()
--. . .


--. . .
NewWebhook.Changed:Wait()
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end

.QueueReached

This event is fired when you reach the rate limit.

:Connect(function(WebhookData: {}))


:Wait() YIELDS


:Once(function(WebhookData: {}))


Example 

local WebhookAPI = require(script.Parent)

local NewWebhook = WebhookAPI.new()

NewWebhook.QueueReached:Connect(function()
    print("Reached rate limit.")     --> {["embeds"] = {. . .}, . . .}
end)

--. . .

NewWebhook:AddEmbed()
--. . .


--. . .
NewWebhook.Changed:Wait()
local success,err = NewWebhook:Send(WebhookURL)

if not success then
    warn(err)
end
